# Bilan et conclusion

Le projet couvre l'ensemble des fonctionnalités de base demandées par le sujet et propose plusieurs extensions cohérentes avec le cahier des charges :
- avis et notes ;
- recommandations ;
- recherche avancée ;
- statistiques évoluées.

Les points forts de cette version sont :
- architecture MVC claire ;
- vue console complète ;
- interface Swing plus ergonomique ;
- code structuré en packages ;
- persistance automatique ;
- gestion explicite des erreurs métier.

Les pistes d'amélioration possibles pour aller encore plus loin seraient :
- playlist collaborative ;
- meilleure mise en forme graphique avec icônes ou thème personnalisé ;
- ajout de tests automatisés JUnit.
